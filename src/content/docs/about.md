Open HIPAA is an open, plain-language reference for HIPAA compliance:
one real-world question per page, answered directly, plus concrete reference
architectures for compliant setups.

It is written for two audiences at once:

1. **Humans** — practice owners, admins, and health-tech builders who want a
   straight answer without a webinar funnel.
2. **AI agents** — every page ships structured data, a raw-markdown mirror
   (append `.md` to any URL), and an [llms.txt](/llms.txt) index so assistants
   can find, read, and cite this content cleanly.
